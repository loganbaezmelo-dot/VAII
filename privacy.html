🔒 Privacy Policy

Effective Date: June 22, 2026

VAII ("the App") is a personal virtual assistant application designed to provide search capabilities and productivity tools. This policy details data management rules inside the interface.

1. Data Usage

VAII processes inputs locally within your browser session.

  • Authentication Profiles: Sign-in records are checked via Firebase Authentication to handle application logs safely.
  • Informational Queries: Parameters regarding weather metrics, search terms, or maps coordinates communicate with standard open channels to print visual updates directly to your screen.

2. Integrated Tools

The interface requests standard metrics from the following services to answer queries:

  • Firebase: For managing user login layouts.
  • Google Cloud Platform: For map views and YouTube data summaries.
  • Open-Meteo: For live timezone and weather readings.
  • Wikipedia / Wiktionary / MyMemory: For information layouts, dictionary checks, and translation strings.
  • Pollinations.ai: For rendering canvas imagery.

3. Local Device Security

Session data remains cached exclusively inside your browser's local memory. Signing out removes these records permanently from your hardware. No continuous tracking systems run on background database servers.

4. Updates

This policy outlines current protocols and adapts alongside functional interface expansions.
