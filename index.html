💡 All Supported Services inside VAII Native: These core operations process natively via external live integrations:
  • 🎤 Voice Recognition: Tap the microphone icon to dictate commands directly.
  • 📰 News Aggregator: Type news about [topic] or top news for live global headlines.
  • 🎬 Media Search: Type movie [title] to pull posters, plots, and IMDB scores.
  • 📝 To-Do Notes: Type note: [text] to save entries, and show notes to manage them.
  • 🧠 Knowledge Base: Input a single term to fetch definitions via Wiktionary.
  • 🔴 YouTube Metrics: Search for famous creators to fetch live stats.
  • 👁️ Gemini Vision: Tap the camera to upload an image and prompt a visual analysis.
  • 🗺️ Maps & Weather: Type map of [city] or weather in [city] for live spatial cards.
  • 🌐 Web Link Routing: Input domains like wikipedia.org or open youtube to launch them.
  • 🍔 Local Food Ordering: Type order [food/brand] (e.g., order McDonald's) for DoorDash routing.
  • 🔢 Calculations & Conversions: Type math or conversions like 193 lbs to kg.
  • 🔄 Translations: Enter Hello to Spanish to convert text.
  • 📈 Market Trackers: Input stock symbols or crypto tickers for live pricing.
  • 🎨 AI Art Generator: Type draw [prompt] to compile custom artwork.
⚠️ CRITICAL RULE: The Gemini Ecosystem mode is an isolated text environment and cannot execute any of the above web widgets.
